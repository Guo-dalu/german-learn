## 三种性

德语每个名词都有性别：**阳性**、**阴性**或**中性**。冠词会告诉你是哪种。

| 性别 | 冠词 | 例子 |
|---|---|---|
| 阳性 | **der** | der Mann（男人）|
| 阴性 | **die** | die Frau（女人）|
| 中性 | **das** | das Kind（孩子）|
| 复数（通用）| **die** | die Leute（人们）|

必须把冠词和名词一起记——没有适用所有词的规律。

## 定冠词（the）

| | 阳性 | 阴性 | 中性 | 复数 |
|---|---|---|---|---|
| **主格**（主语）| der | die | das | die |
| **宾格**（直接宾语）| **den** | die | das | die |
| **与格**（间接宾语 / *mit, von, zu…* 后）| **dem** | **der** | **dem** | **den** |

只有阳性在宾格中变化：**der → den**。

*Der Mann kauft das Buch.* — 主语是 *der Mann*，宾语是 *das Buch*。
*Ich sehe den Mann.* — Mann 是宾语 → **den**。

## 不定冠词（a / an）

| | 阳性 | 阴性 | 中性 |
|---|---|---|---|
| **主格** | ein | eine | ein |
| **宾格** | **einen** | eine | ein |
| **与格** | **einem** | **einer** | **einem** |

不定冠词没有复数形式。表示"没有/不"用 **keine**：*Ich habe keine Zeit.*（我没有时间。）

*Das ist ein Hund.* — 主格，阳性 → **ein**。
*Ich sehe einen Hund.* — 宾格，阳性 → **einen**。

## Kein / Keine（没有）

与 *ein* 的变化规律相同，加前缀 **k-**：

| | 阳性 | 阴性 | 中性 | 复数 |
|---|---|---|---|---|
| **主格** | kein | keine | kein | keine |
| **宾格** | **keinen** | keine | kein | keine |
| **与格** | **keinem** | **keiner** | **keinem** | **keinen** |

*Ich habe keinen Bruder.* — 阳性宾格。
*Das ist keine gute Idee.* — 阴性主格。

## 性别规律

不能每次都猜出来，但这些规律很可靠：

| 词尾 | 性别 | 例子 |
|---|---|---|
| **-ung** | die | die Wohnung, die Zeitung |
| **-heit / -keit** | die | die Freiheit, die Möglichkeit |
| **-tion / -sion** | die | die Station, die Information |
| **-chen / -lein** | das | das Mädchen, das Fräulein |
| **-ment** | das | das Dokument, das Instrument |
| **-er**（表示人）| der | der Lehrer, der Bäcker |
| 星期 / 月份 / 季节 | der | der Montag, der Januar, der Sommer |

## 常见错误

- ❌ *Ich sehe der Mann.* → ✅ *Ich sehe **den** Mann.*（宾格！）
- ❌ *Das ist eine Kind.* → ✅ *Das ist **ein** Kind.*（中性 → *ein*）
- ❌ *Er hat ein Bruder.* → ✅ *Er hat **einen** Bruder.*（宾格阳性）
- ❌ *Das Mädchen ist schön. **Sie** ist meine Schwester.* → ✅ ***Es** ist meine Schwester.*（das Mädchen → *es*）
